 Student Portal

Project Description

This project is a structured and accessible **Student Portal web page** developed using **Semantic HTML5**. It demonstrates the use of modern HTML elements to create a clean, well-organized, and user-friendly layout. The implementation follows best practices in web development, ensuring readability, accessibility, and proper document structure.

 Features

* Semantic HTML5 structure using header, main, section, article, and footer
* Navigation bar with multiple links
* Hero section with welcome message and call-to-action button
* Courses section displaying course details using reusable article components
* Clean and simple layout for better user experience

 Technologies Used

* HTML5

 Project Structure

* `index.html` → Main webpage file containing the complete structure

 Validation

The HTML code has been validated using **W3C Validator** to ensure error-free implementation and adherence to web standards.

 Learning Outcome

* Understanding of Semantic HTML elements
* Importance of structured web design
* Improved accessibility and SEO practices
* Writing clean and maintainable code
